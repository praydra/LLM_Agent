# QMS 불량 처리 및 LOT 보류 기준

## 목적
검사 결과가 NG로 판정된 LOT에 대해 출하 보류, 재검사, 재작업, 원인 분석 기준을 일관되게 적용한다.

## 불량 코드 기준
- DEF-02: 성형 외관 불량. 금형 오염, 원재료 수분, 온도 조건을 우선 확인한다.
- DEF-07: 치수 편차. 성형 조건, 설비 온도, 검사 장비 보정 상태를 함께 확인한다.
- DEF-11: 코팅 두께 편차. 코팅 압력, 점도, 검사 카메라 조명 편차를 확인한다.

## LOT 보류 기준
1. 검사 결과가 NG인 LOT은 자동으로 출하 보류 대상으로 지정한다.
2. HIGH 등급 설비 알람이 동일 공정에서 발생한 경우 영향 범위 LOT을 전후 2시간까지 확대한다.
3. 재작업 LOT은 재검사 OK 판정 전까지 정상 LOT으로 전환하지 않는다.
4. 품질 담당자는 원인, 조치, 재검사 결과를 QMS 이력에 기록한다.

## 답변 시 포함할 항목
- LOT ID와 불량 코드
- 발생 공정과 설비 ID
- 관련 알람 코드와 비가동 시간
- 보류 또는 재작업 필요 여부
- 근거 문서와 조회 조건
