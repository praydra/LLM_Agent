# 설비 알람 조치 표준서

## 목적
설비 알람 발생 시 작업자가 동일한 기준으로 원인을 확인하고, 안전한 상태에서 생산을 재개하도록 한다.

## 공통 조치 절차
1. 알람 발생 시 설비를 안전 정지 상태로 전환한다.
2. MES 알람 이력과 설비 HMI 메시지를 비교해 알람 코드와 발생 시간을 확인한다.
3. 동일 LOT에서 반복 알람이 발생한 경우 품질 담당자에게 LOT 보류 여부를 확인한다.
4. 조치 내용과 재가동 시간을 설비 알람 이력에 기록한다.

## ALM-101 과열 알람
- 주요 원인: 냉각수 유량 저하, 필터 막힘, 온도 센서 오동작
- 1차 조치: 냉각수 밸브 개방 상태와 유량계를 확인한다.
- 2차 조치: 필터를 점검하고 필요 시 교체한다.
- 재가동 기준: 온도가 기준 범위로 복귀하고 10분 이상 안정 상태가 유지되어야 한다.

## ALM-205 압력 저하 알람
- 주요 원인: 압축공기 공급 불안정, 레귤레이터 설정 오류, 배관 누설
- 1차 조치: 메인 압력과 장비 입력 압력을 비교한다.
- 2차 조치: 누설음, 피팅 풀림, 레귤레이터 설정값을 확인한다.
- 재가동 기준: 입력 압력이 기준 하한 이상으로 회복되고 시험 운전 결과가 정상이어야 한다.

## ALM-307 진동 초과 알람
- 주요 원인: 축 베어링 마모, 이송부 체결 불량, 금형 정렬 불량
- 1차 조치: 즉시 생산을 정지하고 설비 보전 담당자에게 점검을 요청한다.
- 2차 조치: 알람 발생 전후 2시간 내 생산된 LOT을 영향 범위 후보로 지정한다.
- 재가동 기준: 진동 값이 기준 범위로 복귀하고 무부하 시험 운전 결과가 정상이어야 한다.

## ALM-402 검사 카메라 알람
- 주요 원인: 렌즈 오염, 조명 편차, 기준 이미지 불일치
- 1차 조치: 렌즈와 조명 모듈을 청소하고 기준 이미지를 재검증한다.
- 2차 조치: 직전 검사 LOT의 불량 코드와 이미지 로그를 함께 확인한다.
- 재가동 기준: 기준 샘플 5개 연속 검사 결과가 정상이어야 한다.

## ALM-510 컨베이어 걸림 알람
- 주요 원인: 이송 가이드 위치 불량, 제품 걸림, 센서 감지 오류
- 1차 조치: 제품과 작업자 안전을 확인한 후 걸림 부위를 제거한다.
- 2차 조치: 이송 가이드 위치와 센서 감지 상태를 확인한다.
- 재가동 기준: 공회전 3회 이상 정상 이송을 확인해야 한다.
