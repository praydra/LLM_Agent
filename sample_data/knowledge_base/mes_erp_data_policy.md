# MES ERP 데이터 조회 및 권한 기준

## 조회 원칙
LLM Agent는 운영 DB에 임의 SQL을 직접 실행하지 않고 사전에 승인된 업무 Tool을 통해서만 데이터를 조회한다.

## MES 조회 범위
- 생산 실적: 일자, 라인, 품목 기준으로 조회한다.
- LOT 이력: LOT ID 기준으로 공정, 설비, 작업자, 검사 결과를 조회한다.
- 설비 알람: 설비 ID 또는 알람 코드 기준으로 알람, 조치, 비가동 시간을 조회한다.

## ERP 조회 범위
- 자재 재고: 자재 코드와 창고 기준으로 가용 수량, 예약 수량, 할당 가능 수량을 조회한다.
- 재고 부족 판단은 최신 업데이트 시각과 생산 계획을 함께 검토해야 한다.

## 보안 및 감사
- 사용자 권한에 따라 조회 가능한 라인, 품목, 창고를 제한해야 한다.
- 모든 Tool 호출은 사용자 ID, 조회 조건, 조회 시각, 응답 요약을 감사 로그로 남겨야 한다.
- 개인정보, 영업비밀, 원가 정보는 마스킹 정책을 적용해야 한다.
