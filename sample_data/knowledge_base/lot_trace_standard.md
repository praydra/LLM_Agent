# LOT 추적 업무 표준

## LOT 추적 기준
LOT 이력은 원재료 투입, 주요 공정 시작과 종료, 검사 결과, 설비 번호, 작업자 정보를 포함해야 한다.

## 품질 이상 LOT 처리
1. 불량 코드가 기록된 LOT은 즉시 출하 가능 상태에서 제외한다.
2. 동일 설비에서 전후 2시간 이내 생산된 LOT을 영향 범위 후보로 지정한다.
3. 품질 담당자는 검사 결과와 설비 알람 이력을 함께 확인한다.
4. 원인 조사가 완료되기 전까지 LOT 상태를 임의로 정상 전환하지 않는다.

## 답변 시 확인해야 할 항목
- LOT ID와 조회 기간
- 공정별 시작 및 종료 시간
- 설비 ID와 작업자
- 검사 결과와 불량 코드
- 관련 설비 알람 및 조치 이력
